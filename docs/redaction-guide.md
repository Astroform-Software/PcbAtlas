# Redaction Guide

PCB data can reveal unreleased products, customer relationships, supplier details, and manufacturing constraints. Before posting in this public repository, remove anything you are not allowed to disclose.

## Do Not Post Publicly

- Complete private PCB source files or project archives.
- Customer, supplier, vendor, or manufacturer names.
- Board names, project names, revision names, serial numbers, or internal ticket IDs.
- Net names, component references, labels, coordinates, stackup notes, manufacturing notes, or pricing details if confidential.
- API keys, AI provider keys, license data, account details, or payment information.
- Screenshots that show private folder paths, user names, customer names, or confidential board content.

## Useful Public Alternatives

When possible, share one of these instead:

- A minimal public sample that reproduces the problem.
- A redacted screenshot showing only the visible symptom.
- Copied error text with private paths and names replaced.
- A short screen recording with sensitive names hidden.
- A description of the data shape, such as "four-layer KiCad board with blind vias" or "ODB++ zip generated by vendor CAM export".
- A link to public vendor documentation or a public sample file.

## How To Describe Private Reproductions

If the issue only reproduces with confidential data, open the issue without attaching the file. Include:

1. PCB Atlas version and build number.
2. Platform, device, and OS version.
3. File format and import/open path.
4. Visible symptom.
5. Whether the issue is repeatable.
6. A high-level description of the board or package structure.

A maintainer can decide whether a private transfer path is needed.

## Redaction Checklist

Before submitting, check:

- The issue title contains no private project or customer name.
- Screenshots have no private board text, paths, email addresses, or account names.
- Logs and Support Bundle text do not include secrets or private file paths.
- Attachments are either public samples or files you are allowed to share publicly.
- Any AI provider, license, payment, or account data has been removed.
