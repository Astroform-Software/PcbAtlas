# Support

Use GitHub issues in this repository for public, non-confidential feedback and bug reports.

## Public Support

Open an issue when the report can be discussed publicly without exposing private PCB data.

Useful public reports include:

- App crashes or hangs.
- Import failures with a public or minimal sample.
- Rendering differences that can be shown with redacted screenshots.
- Documentation corrections.
- General feature requests.

## Private Data

Do not upload confidential board files to public issues. If a report requires private files, open a public issue with a high-level description and wait for a maintainer to provide a safe sharing path.

For direct contact, use the public PCB Atlas email listed on the website: <pcbatlas@astroformsoftware.com>.
