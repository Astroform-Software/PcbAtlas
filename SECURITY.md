# Security Policy

Please do not report security issues, private keys, license problems, or confidential PCB data in public GitHub issues.

## Sensitive Information

PCB files and support materials may include:

- Customer or supplier names.
- Unreleased product designs.
- Net names, component names, board dimensions, or stackup details.
- Manufacturing notes, pricing, or commercial terms.
- AI provider keys or account tokens.

Do not upload this information publicly.

## Reporting

For now, open a public issue only with a non-sensitive summary, then wait for a maintainer to provide a private path if one is needed.

If you accidentally shared sensitive content, delete the attachment or issue comment immediately and notify a maintainer in the issue without repeating the secret.
